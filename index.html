Built-in AI Code Generator Integration

Cainban includes a built-in MCP (Model Context Protocol) server that connects seamlessly with AI code generators like Claude, ChatGPT, and other AI assistants. Just run cainban mcp and configure your AI tool - no complex setup or additional servers required.

🤖 Natural Language Commands

"Create a high priority task for user authentication" → Executes CLI commands through your AI assistant

🔄 Real-time Sync

Changes made through AI assistants appear immediately in your terminal

📋 Smart Task Management

Link tasks, set priorities, and organize boards through conversation with any compatible AI

AI Assistant + Cainban
You: Create a task for implementing the login feature with high priority
AI: Created task #3 [HIGH]: Implement login feature
Added to your "Development" board in TODO column
You: Link task 3 to block task 1
AI: Linked task 3 blocks task 1
Task 1 is now blocked until task 3 is completed

Setup AI Integration

1. Start the MCP Server

./cainban mcp

Launches the built-in MCP server on your local machine

2. Configure Your AI Assistant

ai-tool mcp add cainban '{"command":"./cainban","args":["mcp"]}'

Connects your AI code generator to Cainban MCP server

3. Start Managing Tasks

"Add a task for user authentication with high priority"

Use natural language to manage your tasks through any compatible AI

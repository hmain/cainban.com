Setup MCP Integration

1. Start the MCP Server

./cainban mcp

Launches the built-in MCP server on your local machine

2. Configure Claude

claude mcp add-json cainban '{"type":"stdio","command":"./cainban","args":["mcp"]}'

Connects Claude to your Cainban MCP server

3. Start Chatting

"Add a task for user authentication with high priority"

Use natural language to manage your tasks through Claude

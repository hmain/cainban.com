Setup AI Codegen Workflow

1. Start the MCP Server

./cainban mcp

Launches the built-in MCP server on your local machine

2. Connect Your AI Development Tools

ai-tool mcp add cainban '{"command":"./cainban","args":["mcp"]}'

Links all AI assistants to unified project management workflow

3. Begin AI-Enhanced Development

"Add a task for user authentication with high priority"

Transform development conversations into structured, trackable project workflows
